Built for people who just want it to work

Every feature is included by default. No premium tier, no upsells — just a solid torrent client.

⚡

Magnets & .torrent

Add torrents by magnet link or file. VoidTorrent resolves peers through trackers, DHT, PEX, and local discovery.

📂

File & priority picker

Open a torrent and tick only the files you want. Set high, normal, or skip priority per file before and during transfer.

Sequential download

Download in order so you can preview media while the rest finishes — perfect for large video packs.

Cap upload/download, limit connections, and tune active queues so torrents never hog your connection.

FAQ

Quick answers before you install.

Is it really free?

Yes. VoidTorrent is open-source and free forever. No paid version, no ads.

Windows only?

v1.0 targets Windows 10/11 (x64). The libtorrent engine is cross-platform and ports are planned.

Is it safe?

The code is transparent and built on libtorrent. Always download from this site and verify the checksum.

Does it seed?

By default torrents are auto-managed and keep seeding after completion until you remove them.
