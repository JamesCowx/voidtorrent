How to use VoidTorrent

From download to first finished file in five minutes. Follow the steps below.

Download and run the installer

Get VoidTorrent-1.0.0-x64.exe from the download page. It's a single self-contained executable — double-click to launch. No setup wizard, no admin prompts.

Add your first torrent

Click Add Torrent File to pick a .torrent, or Add Magnet to paste a magnet:?xt=urn:btih:... link. In the dialog you can choose where to save and tick Download sequentially for media you want to preview early.

Pick the files you want

Open the Details tab, right-click any file, and set its priority. Choose Skip file to exclude it entirely — great for skipping sample clips or bonus folders inside a big pack.

Control transfers

Use Pause / Resume / Remove on the toolbar or right-click a torrent. Removing asks whether to also delete the downloaded data, so nothing is lost by accident.

Tune speed & connection

Open Options to set global download/upload limits, the listen port, and toggle DHT / UPnP / NAT-PMP. Changes apply live. Close the window and VoidTorrent keeps running in the system tray.

Tips: VoidTorrent resumes unfinished downloads automatically after a restart. Keep the app running to keep seeding and help others. Set an upload limit if you're on a metered connection.
